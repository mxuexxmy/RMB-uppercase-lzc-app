# 人民币大写转换工具

一个简洁美观的人民币大写转换工具网站，将阿拉伯数字金额转换为中文大写金额。

## 功能特点

- 🎯 **准确转换**：支持0-9999亿的金额转换
- 💰 **完整格式**：支持元、角、分的完整转换
- 🎨 **现代界面**：采用渐变背景和卡片式设计
- 📱 **响应式**：完美适配桌面和移动设备
- ⚡ **实时验证**：输入时自动验证格式
- 🎪 **交互体验**：支持回车键转换和示例点击

## 技术栈

- **HTML5** - 语义化标签和现代HTML特性
- **CSS3** - 渐变背景、动画效果、响应式设计
- **JavaScript ES6+** - 面向对象编程、事件处理

## 使用方法

1. 打开 `index.html` 文件
2. 在输入框中输入金额（支持小数，最多2位小数）
3. 点击"转换"按钮或按回车键
4. 查看转换结果

## 转换规则

- 支持整数和小数金额
- 小数部分最多支持2位（角、分）
- 整数部分按万、亿分组处理
- 自动处理零的显示规则
- 金额为0时显示"零元整"
- 无小数时显示"整"

## 示例

| 输入 | 输出 |
|------|------|
| 1234.56 | 壹仟贰佰叁拾肆元伍角陆分 |
| 1000000 | 壹佰万元整 |
| 0.01 | 壹分 |
| 0 | 零元整 |
